# DicomToolboxMatlab
DicomToolboxMatalb allows to import, visualize, and extract image features from CT and RT Dose DICOM files in MATLAB.

## Import
* CT
* RT Dose
* RT Structures
* Grid interpolation

## Visualization
* CT
* Dose
* Dose-volume histogram

## Feature extraction
* Organ shape
  * volume
  * area
  * sphericity
  * eccentricity
  * compactness
  * density
  * eigenvalues
* Dose-volume histogram
  * mean
  * spread
  * skewness
  * kurtosis
  * Dx
  * Vx
  * entropy
  * uniformity
* Subvolume dose
* Dose shape
  * Dose gradients
  * Dose moments
